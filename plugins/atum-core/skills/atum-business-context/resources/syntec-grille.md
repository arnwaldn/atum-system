## Duree du travail

### Forfait jours (cadres IC)
- **218 jours/an** maximum (forfait cadre Syntec)
- Equivalent 35h hebdomadaires avec RTT
- RTT : environ 10-12 jours/an selon les annees

### Temps de travail effectif
| Element | Jours |
|---|---|
| Jours calendaires | 365 |
| Weekends | -104 |
| Jours feries | -8 (moyenne) |
| Conges payes | -25 |
| RTT | -10 (environ) |
| **Jours travailles** | **~218** |

### Heures supplementaires (ETAM)
- 25% de majoration pour les 8 premieres heures (36h-43h)
- 50% de majoration au-dela (>43h)

## Conges et absences

### Conges payes
- **25 jours ouvrables/an** (5 semaines)
- Periode de reference : 1er juin — 31 mai
- Jours de fractionnement : +1 jour si 3-5 jours CP hors ete, +2 jours si 6+ jours

### Conges exceptionnels (Syntec)
| Evenement | Jours |
|---|---|
| Mariage du salarie | 4 |
| Naissance / Adoption | 3 |
| Mariage d'un enfant | 1 |
| Deces conjoint/partenaire | 3 |
| Deces parent (pere/mere) | 3 |
| Deces enfant | 5 |
| Demenagement | 1 |

### Maladie
- Maintien de salaire selon anciennete (apres 1 an)
- Complement employeur apres carence de 3 jours

## Charges sociales (estimation)

### Cout employeur = Brut x 1.45-1.55

| Charge | Part employeur |
|---|---|
| Securite sociale (maladie, vieillesse, AF) | ~30% |
| Retraite complementaire (AGIRC-ARRCO) | ~10% |
| Assurance chomage | ~4.05% |
| Formation professionnelle | ~1.68% |
| Taxe apprentissage | ~0.68% |
| Prevoyance + mutuelle | ~3-5% |
| **Total employeur** | **~45-55%** |

### Formule de calcul rapide
```
cout_total_employeur = salaire_brut * 1.50
cout_total_annuel = cout_total_employeur * 12 (ou 13 si 13e mois)
tjm_interne = cout_total_annuel / 218
```

### Exemple
```
Dev confirme 45K brut/an
Cout employeur : 45K * 1.50 = 67.5K EUR/an
TJM interne : 67 500 / 218 = ~310 EUR/jour
Marge sur TJM facture 550 EUR : (550-310)/550 = 43.6%
```

## Prevoyance et mutuelle

### Prevoyance cadres (obligatoire Syntec)
- Cotisation minimum : 1.50% tranche A (50% employeur)
- Garanties : deces, incapacite, invalidite

### Mutuelle (obligatoire depuis 2016)
- 50% minimum pris en charge par l'employeur
- Panier de soins minimum ANI
- Portabilite : maintien 12 mois apres depart

## BSPCE — Bons de Souscription de Parts de Createur d'Entreprise

### Conditions eligibilite ATUM
- SAS de moins de 15 ans
- Non cotee
- Beneficiaires : salaries et mandataires sociaux (president, DG)
- Detention minimale 25% par personnes physiques

### Fiscalite
- Plus-value : flat tax 30% (12.8% IR + 17.2% PS)
- Si detention < 3 ans : regime salaires (plus defavorable)

### Implementation prevue An2
- Pool BSPCE : 5-10% du capital post-dilution
- Vesting : 4 ans avec cliff 1 an
- Exercice : au pair (1 EUR/action) ou a valeur de marche
